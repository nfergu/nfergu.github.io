Selected Open Source Contributions

  • A utility for performance instrumentation of Spark jobs. Currently part of the Big Data Genomics Utils package. See this pull request and this pull request for the original pull requests. See the "Instrumentation" section of the BDG Utils documentation for detailed documentation (Scala).
  • Re-write of the duplicate-marking code in the ADAM genomics analysis platform, making it 3x faster. See this pull request (Scala).
  • Modifications to the Base Quality Score Recalibration code in the ADAM genomics analysis platform, to improve performance by around 2x. See this pull request and this pull request (Scala).
  • An example of population stratification analysis based on Spark and Deep Learning. See this Github repo, and the companion blog post.
  • An enhancement for Spark to support looking-up of named accumulators in a registry (not merged). See this pull request (Scala).
  • An analysis of hash table performance in 3 languages: C++, Java, and Python. See this Github repo (C++, Java, Python).
  • A simple utility for measuring the performance of blocks of C++ code. See this Github repo (C++).
